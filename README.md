# Navegame.py: o jogo das naves

Jogo simples de nave feito em Python utilizando a biblioteca Pygame. O desafio do jogo é o desvio de objetos celestiais com uma nave redonda, havendo a contagem de pontos por desvios bem sucedidos e um sistema de recorde dos maiores pontos.

O jogo utiliza gráficos minimalistas representando a nave e o jogador com formas circulares e controles simples, servindo de introdução à programação de jogos e pensando na utilização em consoles caseiros ou educacionais.

## Dependências

Para inicialização é necessário ter instalado o [Python](https://wiki.python.org/moin/BeginnersGuide/Download) e o [Pygame](https://www.pygame.org/wiki/GettingStarted).

## Rodando o jogo

É recomendado rodar pelo terminal, dentro do diretório do jogo e chamando com `python`:

```
$ python Navegame.py.pygame
```

### Batocera

Para jogar no Batocera, criar o diretório do jogo dentro de `/userdata/roms/pygame/` e atualizar a lista de jogos, para rodar diretamente, [como no tutorial da documentação oficial](https://wiki.batocera.org/systems:pygame), pois o Batocera já vem com pygame nativo.
